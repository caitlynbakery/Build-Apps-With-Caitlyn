# BUILD APPS WITH CAITLYN


![](images/phones.png)

Android app to show Caitlyn's courses.
I used these technologies:

* Java
* [Picasso](https://square.github.io/picasso/)
* Palette
* RecyclerView

## USAGE

install apk in android phone version 6 or higher 

## ENVIRONMENT

* Android 6.1
* gradle 4
* Android SDK 28
